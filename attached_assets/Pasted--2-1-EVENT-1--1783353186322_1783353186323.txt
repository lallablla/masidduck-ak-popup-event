기존 랜딩페이지 구조를 아래 이벤트 정책에 맞게 수정해줘.

이벤트는 “혜택 2단계 구조”로 운영한다.

1. 메인 이벤트: 인스타 인증 EVENT
혜택: 피자설기 1개 현장 증정

이 이벤트를 페이지 상단 메인 영역에 가장 먼저 보여줘.
고객이 QR로 들어왔을 때 가장 먼저 “인스타 인증하면 피자설기 1개 증정”이 보이게 해줘.

참여 조건:
- 마시떡 인스타그램 팔로우
- 팝업 현장 사진 또는 피자설기 사진을 스토리/게시물로 업로드
- @masidduck 계정 태그
- 필수 해시태그 추가: #피자설기 #AK수원팝업
- 참여 완료 화면을 직원에게 보여주기

메인 문구:
인스타 인증하고
피자설기 1개 받아가세요!

설명 문구:
마시떡 인스타그램을 팔로우하고
팝업 현장 사진 또는 피자설기 사진을 올려주세요.

필수 태그:
@masidduck
#피자설기 #AK수원팝업

참여 후 완료 화면을 직원에게 보여주시면
피자설기 1개를 드립니다.

2. 플러스 이벤트: 베스트 리뷰 EVENT
혜택: 피자설기 8개입 자택 배송

인스타 인증 이벤트 참여자 중
정성스러운 사진과 후기를 남겨주신 분을 선정해
피자설기 8개입을 자택으로 보내드립니다.

문구:
PLUS EVENT
베스트 리뷰 선정 시
피자설기 8개입 자택 배송!

설명:
인스타 인증 이벤트 참여자 중
정성스러운 사진과 후기를 남겨주신 분께
팝업 종료 후 피자설기 8개입을 보내드립니다.

당첨 안내:
선정되신 분께는 팝업 종료 후
인스타그램 DM으로 개별 연락드립니다.

주의 문구:
※ 계정이 비공개인 경우 확인이 어려울 수 있습니다.
※ DM 미응답 시 선정이 취소될 수 있습니다.
※ 당첨자 선정 및 배송 안내는 팝업 종료 후 개별 연락드립니다.

3. 서브 이벤트: 빠른 참여 EVENT
혜택: 증정떡 1개 또는 현장 혜택
단, 이 혜택 문구는 관리자 모드에서 수정할 수 있게 해줘.

기본 문구는:
빠른 참여도 OK!
아래 중 하나만 참여해도 증정 혜택을 드려요.

참여 조건:
- 카카오톡 채널추가
- 네이버 저장·알림받기
- 유튜브 구독
중 하나만 참여

참여 후 완료 화면을 직원에게 보여주기

버튼 순서:
1. 인스타그램 팔로우하기
2. 카카오톡 채널추가하기
3. 네이버 저장·알림받기
4. 유튜브 구독하기

다만 페이지 상단에서는 인스타 인증 이벤트를 가장 크게 보여주고,
빠른 참여 이벤트는 그 아래 보조 이벤트로 보여줘.

4. 해시태그 안내
인스타 인증 이벤트 영역에 필수 태그와 추천 태그를 구분해서 보여줘.

필수 태그:
@masidduck
#피자설기 #AK수원팝업

추천 태그:
#수원맛집 #수원디저트 #아이간식 #쌀디저트 #K디저트 #이색디저트

필수 태그는 고객이 꼭 넣어야 하는 것으로 표시하고,
추천 태그는 “함께 달면 좋아요” 정도로 작게 보여줘.

5. 관리자 모드 수정 항목 추가
관리자 비밀번호 0710 입력 후 아래 항목도 모바일에서 수정 가능하게 해줘.

- 메인 이벤트 진행 여부
- 인스타 인증 이벤트 혜택 문구
  예: 피자설기 1개 증정
- 빠른 참여 이벤트 진행 여부
- 빠른 참여 이벤트 혜택 문구
  예: 증정떡 1개 / 1,000원 할인 / 이벤트 일시 중단
- 베스트 리뷰 이벤트 진행 여부
- 베스트 리뷰 혜택 문구
  예: 피자설기 8개입 자택 배송
- 오늘의 공지 문구
- 강조 안내 문구
- 각 버튼 노출 여부
  인스타그램 / 카카오톡 / 네이버 / 유튜브

6. 이벤트 상태별 화면
관리자 모드에서 이벤트 상태를 바꿀 수 있게 해줘.

상태 옵션:
- 전체 이벤트 진행 중
- 인스타 인증 이벤트만 진행 중
- 빠른 참여 이벤트만 진행 중
- 증정품 소진
- 이벤트 일시 중단

각 상태에 따라 고객 화면 상단 공지 박스 문구가 눈에 띄게 바뀌어야 해.

예시:
증정품 소진 상태일 때:
현재 준비된 증정품이 모두 소진되어 이벤트가 일시 종료되었습니다.
많은 관심 감사합니다.

빠른 참여 이벤트만 진행 중일 때:
현재 현장 상황으로 인해 빠른 참여 이벤트만 진행 중입니다.
카카오톡 채널추가 / 네이버 저장·알림받기 / 유튜브 구독 중 하나를 참여해주세요.

인스타 인증 이벤트만 진행 중일 때:
현재 피자설기 증정 인스타 인증 이벤트를 진행 중입니다.
인스타 팔로우 후 사진 업로드와 태그를 완료해주세요.

7. 고객 화면 디자인 우선순위
모바일 첫 화면에서 아래 순서로 보이게 해줘.

1) 마시떡 AK플라자 수원점 팝업 EVENT
2) 인스타 인증하고 피자설기 1개 받기
3) QR로 들어온 고객이 바로 누를 수 있는 인스타그램 버튼
4) 필수 태그 안내
5) 베스트 리뷰 플러스 이벤트
6) 빠른 참여 이벤트
7) 유의사항

8. 전체 톤
기존처럼 따뜻한 베이지와 딥레드 포인트를 유지해줘.
너무 쿠폰 이벤트처럼 보이지 않게 하고,
백화점 팝업에 어울리는 깔끔하고 정갈한 디자인으로 만들어줘.

중요:
이벤트 내용은 현장 상황에 따라 계속 바뀔 수 있으므로,
관리자 모드에서 텍스트와 노출 여부를 쉽게 수정할 수 있어야 해.
관리자 수정 내용은 모든 방문자에게 동일하게 반영되어야 하므로,
Node.js + Express + data.json 저장 방식으로 구현해줘.
localStorage만 사용하는 방식은 안 돼.
